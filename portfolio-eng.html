Disc-ing [Under Development]

A platform where gamers of all kinds can rate their favorite games, giving their opinions and interacting with the community

Disc-ing is a digital library for gamers that want to organize their favorite games, rate them and discovering new experiences throughout the community. Counting on an intuitive interface, the project allowes the users to share opinions, explore other players evaluations and build their own personal collection. A space made for gamers, by gamers.

Administrators have access to an exclusive page where they are able to register new games to the platform. Each title has their own page, where informations about the game story, technical requirements and evaluations by the users can be found.

Calculator

This calculator was a personal project. After the introduction to JavaScript I was still lacking the knowledge that is acquired by practicing. Therefore, I started this project to develop this missing knowledge.

In the begining, since it was my first project using this language, I went through a lot of difficulties. Throughout the days I was able to evolve and implement the functionalities I desired.

After the calculator was finished I decided to work on a currency converter on the same page. The conversor fetches the current value of the currency using a free API and make the conversions based on the user input. There are also themes that can be changed to make it more attractive for the user.

Pokémon Guess Who

As soon as my college vacation begun I knew I wanted to start a new project. Then I remembered the Pokémon API I came across earlier this year, so I decided to use it on a game.

Using the API to fetch the pokémon information and a common ip to get the sprites for each of them I was able to generate nearly infinite "guess Who" game posibilities. To play you must copy the code of your game and send to your oponent, so you both have the same pokemon possibilities to guess.
