Disc-ing [Em Desenvolvimento]

Uma plataforma onde jogadores de todos os tipos podem avaliar seus jogos favoritos, dando suas opiniões e interagindo com a comunidade!

Disc-ing é uma biblioteca digital interativa voltada para gamers que desejam organizar seus jogos favoritos, avaliá-los e descobrir novas experiências através da comunidade. Com uma interface intuitiva, o projeto permite que os usuários compartilhem opiniões, explorem avaliações de outros jogadores e construam sua própria coleção pessoal. Um espaço feito por e para apaixonados por jogos.

Administradores têm acesso a uma área exclusiva para cadastrar novos jogos na plataforma. Cada título possui sua própria página, onde são exibidas informações sobre a história do jogo, requisitos técnicos e as avaliações deixadas pela comunidade.

Calculadora

Minha calculadora foi um projeto pessoal. Após a introdução ao JavaScript ainda me faltava conhecimentos que se adquirem na fase prática, por isso, resolvi iniciar este projeto.

No começo enfrentei várias dificuldades por ser meu primeiro projeto com a linguagem, mas depois dos primeiros dias consegui evoluir e implementar as funções que queria.

Além da calculadora implementei um conversor de moedas na mesma página, que puxa os valores de conversão diários por meio de uma API gratuita e converte os valores conforme o usuário deseja. A página também conta com temas para tornar a página mais agradável para o usuário.

Cara-a-Cara Pokémon

Logo no começo das minhas primeiras férias da faculdade tive vontade de iniciar algum projeto. Foi aí que lemebrei que vi uma API que contém todos os pokémon e pensei em criar um jogo usando-a.

Utilizando uma Api para buscar informações de cada pokémon e outra para buscar os sprites deles consegui criar o jogo com inúmeras possibilidades de tabuleiro. Para jogar é necessário copiar o código do tabuleiro e enviar para o seu oponente, de modo que ambos tenham os mesmos pokémon para adivinhar.
